# Bill

Bill is a demo application.

Try it on Heroku here [https://young-springs-67533.herokuapp.com/](https://young-springs-67533.herokuapp.com/)

## Prerequisite
You will need the following things properly installed on your computer
* Git
* Ruby 2.3.0

## Installation
* `git clone git@gitlab.com:ChrisWilding/bill.git`
* `cd bill`
* `bundle install`

## Running / Development
* `bundle exec rails s`
* Visit [http://localhost:3000/](http://localhost:3000/)

### Running Tests
* `bundle exec rspec`
* `bundle exec rake spec:javascript`
